# Goose Analytics (Prototype)

Prototype of WIP Goose Analytics based on Vue.

## Environment Variables

```
SERVER_PORT=3000
DATABASE_URL=mongodb://localhost:27017/goosedb_preview
```

## Development

First fork this repo and clone it.

### Vanilla Node.js Server

```bash
npm install
npm run serve
```
